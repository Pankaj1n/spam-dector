⚙️ How It Works

Load Data — Dataset loaded using Pandas
Preprocess — Text cleaned and vectorized using TfidfVectorizer
Train Model — Trained using Naive Bayes classifier from Scikit-learn
Evaluate — Accuracy, precision, and confusion matrix generated
Predict — Input any message → Model says SPAM or NOT SPAM
